## 第1章 测试概述

### 1.1 测试背景

GraphHire是基于AI智能解析与知识图谱的招聘匹配系统，本次测试旨在验证系统核心功能是否满足赛题要求的功能规格。

### 1.2 测试范围

| 测试对象 | 测试内容 |
|----------|----------|
| 个人用户端 | 注册登录、简历上传、文档解析、能力图谱、职位推荐、匹配详情 |
| 企业用户端 | 注册登录、企业认证、职位发布、文档上传、能力图谱、候选人推荐 |
| 管理后台 | 用户管理、企业认证审核、简历库、职位库、技能标签、任务监控、日志查询 |

### 1.3 测试环境

| 环境项 | 配置 |
|--------|------|
| 操作系统 | Windows 11 Pro / 银河麒麟V10 |
| 浏览器 | Chrome 120、Firefox 121、Edge 120 |
| 数据库 | PostgreSQL 16 |
| JDK | OpenJDK 21 |
| 前端 | Next.js 14、Ant Design 6.x |

## 第3章 接口测试摘要

### 3.1 认证接口

| 接口 | 方法 | 测试要点 | 预期结果 |
|------|------|----------|----------|
| /auth/sendCode | POST | 发送验证码，60秒内重复发送应被拦截 | 首次成功，60秒内重复提示"发送过于频繁" |
| /auth/register/person | POST | 参数完整性和格式校验 | 符合规则返回200，不符合返回400 |
| /auth/login | POST | 正确/错误密码、连续失败锁定 | 正确登录返回token，5次失败锁定15分钟 |
| /auth/refreshToken | POST | access_token过期后使用refresh_token续期 | 返回新的access_token |

| 接口 | 方法 | 测试要点 | 预期结果 |
|------|------|----------|----------|

| /person/resume/list | GET | 分页、筛选 | 返回当前用户的简历分页列表 |
| /person/graph | GET | 无简历/有简历场景 | 无简历返回空数据，有简历返回图谱数据 |
