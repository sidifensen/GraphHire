# 软件产品说明书

## 第1章 产品概述

### 1.1 产品简介

GraphHire（图谱智聘）是一款基于银河麒麟操作系统的AI智能匹配与能力图谱系统。系统采用B/S架构，通过文档智能解析与知识图谱技术，实现简历与职位要求的深度结构化解析、能力可视化与智能双向推荐。

### 1.2 产品定位

面向省级人才服务中心（10万+份简历、5000+家企业）的智能招聘平台，替代传统人工筛选模式，提升人岗匹配效率与精准度。

### 1.3 目标用户

| 用户类型 | 用户画像 | 核心诉求 |
|----------|----------|----------|
| 个人求职者 | 应届生、转职者 | 快速找到匹配职位，量化自身能力 |
| 企业HR | 招聘专员、企业管理者 | 高效筛选候选人，降低招聘成本 |
| 平台管理员 | 人才中心运维人员 | 系统运维、数据管理、运营监控 |

### 1.4 核心价值

- **效率提升**：AI自动解析替代人工阅读，匹配效率提升10倍+
- **精准度提升**：基于技能图谱的语义匹配，突破关键词匹配局限
- **透明化**：匹配评分有据可依，求职者清晰了解自身优劣势

## 第3章 多维度匹配说明

### 3.1 匹配维度与权重

| 维度 | 默认权重 | 说明 |
|------|----------|------|
| 技能匹配 | 50% | 核心技能的重合度与覆盖度 |
| 经验匹配 | 20% | 工作年限与职位级别的符合度 |
| 城市匹配 | 15% | 意向城市与工作城市的符合度 |
| 学历匹配 | 10% | 学历要求与实际学历的符合度 |
| 薪资匹配 | 5% | 期望薪资与职位薪资范围的符合度 |

### 3.2 城市加分规则

- **居住城市匹配**：若求职者居住城市与职位工作城市相同，额外+5分
- **意向城市匹配**：若求职者意向城市与职位工作城市相同，额外+3分

### 3.3 匹配度分制

综合匹配分采用百分制（0-100分）：

| 分数区间 | 等级 | 说明 |
|----------|------|------|
| 80-100 | A级 | 高度匹配，推荐优先处理 |
| 60-79 | B级 | 较好匹配，可纳入候选 |
| 40-59 | C级 | 基本匹配，需进一步评估 |
| 0-39 | D级 | 匹配度低，不推荐 |

## 第6章 术语表

| 术语 | 说明 |
|------|------|
| 能力图谱 | 以图结构展示个人或职位的技能、能力及其关联关系的可视化形式 |
| 匹配度 | 衡量求职者与职位契合程度的量化指标（0-100分） |
| 技能归一化 | 将不同表述的同义技能（如"Java"和"Java语言"）统一为标准技能标签 |
| 解析状态 | 文档从上传到完成AI解析的过程状态：待解析→解析中→成功/失败 |
| 图谱节点 | 能力图谱中的顶点，代表技能、教育背景、工作经验等实体 |
| 图谱关系 | 能力图谱中的边，代表节点之间的关联（如"掌握"、"要求"） |
