# 参考数据集

## Job-SDF 数据集

**Job-SDF** (Job Skill Demand Forecasting) 是一个用于职位技能需求预测的多粒度基准数据集。

### 基本信息

| 项目 | 说明 |
|------|------|
| 官网 | https://job-sdf.github.io/ |
| GitHub | https://github.com/Job-SDF/benchmark |
| 论文 | https://arxiv.org/abs/2405.11220 |
| 数据下载 | https://github.com/Job-SDF/benchmark/tree/main/dataset |
| 许可证 | ⚠️ 无明确LICENSE（联系作者获取授权） |
| 联系邮箱 | hxzhang@ustc.edu.cn |

### 数据来源

- 中国科学技术大学
- 智联招聘（Career Science Lab）
- 清华大学金融学院
- 北京航空航天大学人工智能研究院
- 香港科技大学（广州）

### 数据格式

| 目录 | 说明 |
|------|------|
| `demand/` | 主需求数据，每行代表一种技能，每列是不同的月份 |
| `proportion/` | 需求比例数据，值为0-1之间的比例 |
| `graph/` | 技能共现图，三元组(skill_id_1, skill_id_2, co-occurrence_frequency) |
| `entity_map/` | 技能/职业/地区名称索引表 |

### 对本项目的参考价值

| 参考方向 | 说明 |
|----------|------|
| **技能分类体系** | 可借鉴其技能层级分类（L1/L2职业分类） |
| **技能共现关系** | 用于技能归一化的辅助参考 |
| **职位分类** | 借鉴职业分类体系 |

### 使用限制

| 限制项 | 说明 |
|--------|------|
| 简历数据 | ❌ 不包含，无法直接用于简历匹配 |
| 用户信息 | ❌ 不包含，只有职位需求数据 |
| 实时性 | ⚠️ 数据为2021-2023年，可能过时 |
| 商业授权 | ⚠️ 未明确，需联系作者 |

### 结论

Job-SDF 数据集主要适用于**技能需求预测研究**，对于本项目的直接用途有限。

**推荐参考**：
1. 技能层级分类体系
2. 技能共现关系图
3. 职位分类方法

**不建议**：
- 直接复制数据到数据库
- 用于商业目的（除非获得授权）
