## 7.4 响应式设计策略

### 适配目标

| 设备类型 | 屏幕宽度 | 适配策略 |
|----------|----------|----------|
| 手机 | < 768px | 移动优先，单列布局 |
| 平板 | 768px - 1024px | 自适应，双列布局 |
| 桌面 | ≥ 1024px | 完整布局，多列展示 |

### 移动端适配原则

1. **触摸优先**
   - 按钮最小点击区域 44x44px
   - 列表项高度 ≥ 48px
   - 表单项间距适当增大

2. **单手操作**
   - 常用操作放在屏幕下半部分
   - 底部固定操作栏（如保存/提交）
   - 顶部返回按钮 + 标题

3. **简化导航**
   - 汉堡菜单替代顶部横向导航
   - 底部 Tab 栏（个人端常用）
   - 抽屉式侧边栏（企业端/管理后台）

4. **卡片化布局**
   - 信息以卡片为单位展示
   - 卡片宽度占满屏幕
   - 卡片之间垂直排列

## 7.5 主题模式

### 支持模式

| 模式 | 说明 | 触发方式 |
|------|------|----------|
| 浅色模式 | 默认主题 | - |
| 深色模式 | 夜间使用 | 手动切换 + 系统同步 |
| 自动模式 | 跟随系统设置 | - |

### 主题切换组件

- 位置：Header 右侧
- 图标：☀️ 浅色 / 🌙 深色 / 📱 自动
- 持久化：LocalStorage 存储用户偏好
