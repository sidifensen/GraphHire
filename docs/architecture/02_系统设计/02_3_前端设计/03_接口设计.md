## 8.9 通用响应格式

### 成功响应

```json
{
  "code": 200,
  "msg": "success",
  "data": { ... }
}
```

### 分页响应

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "list": [...],
    "total": 100,
    "page": 1,
    "size": 10
  }
}
```

### 错误响应

```json
{
  "code": 400,
  "msg": "错误描述",
  "data": null
}
```

## 8.10 认证说明

- **是**：需要携带有效 Token（通过 Sa-Token 实现）
- **否**：无需认证即可访问

### Token 传递方式

- Header: `Authorization: Bearer <token>`

### 通知类型 (NotificationType)

| 值 | 说明 |
|----|------|
| 1 | 系统通知 |
| 2 | 匹配通知 |
| 3 | 简历被查看 |
| 4 | 职位申请 |
| 5 | 简历被查看（企业端） |
