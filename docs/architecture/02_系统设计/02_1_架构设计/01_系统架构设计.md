## 3.2 技术栈明细

| 层级 | 技术选型 | 版本 | 说明 |
|------|----------|------|------|
| 前端 | Next.js | 16.2.3 | App Router, React Server Components, Turbopack |
| 前端框架 | React | 19.2.5 | 最新并发特性 |
| 前端CSS | TailwindCSS | 4.x | 原子化CSS框架 |
| 前端UI | Ant Design | 6.3.5 | PC管理后台 |
| 前端图表 | Ant Design Charts（@ant-design/plots） | 统计图表（折线、柱状、饼图等） |
| 前端图谱 | Ant Design Charts（@ant-design/graphs） | 知识图谱渲染 |
| 后端框架 | Spring Boot 3 + DDD | 3.4.5 | 领域驱动设计，JDK 21 |
| 认证 | sa-token | 1.45.0 | 轻量级权限框架 |
| ORM | MyBatis-Plus | 3.5.10 | PostgreSQL访问 |
| 任务队列 | RocketMQ | 5.5.0 | 异步解析任务 |
| 文档解析 | Apache Tika | 3.2.0 | PDF/DOC文本提取 |
| 智能提取 | 大模型API（如DeepSeek-V3.2） | - | 结构化实体提取 |
| 图数据库 | Neo4j | 5.26.12 | 知识图谱存储 |
| 文件存储 | RustFS | 1.0.0-alpha.93 | S3兼容对象存储 |
| 缓存 | Redis | 8.6.2 | 会话缓存热点数据 |

## 3.3 架构分层详述

| 层级 | 技术/组件 | 职责说明 |
|------|----------|----------|
| **用户层** | 个人用户（PC Web/H5）、企业用户（PC Web/H5）、管理员（PC Web） | 三类角色通过浏览器访问系统，各自操作对应功能模块 |
| **接入层** | Next.js + TailwindCSS + Ant Design + Ant Design Charts | 响应式前端界面，个人端展示能力图谱/职位推荐，企业端展示候选人推荐，管理员后台进行系统管理 |
| **服务层** | Nginx + sa-token | 统一路由转发、认证鉴权 |
| **业务层** | Spring Boot 3 + DDD + sa-token + MyBatis-Plus + RocketMQ | 用户认证（注册/登录）、文档上传与存储（RustFS）、任务调度（解析队列）、图谱构建（Neo4j）、匹配推荐、后台管理，DDD架构包含：领域模型、聚合根、领域服务、仓库接口、应用服务 |
| **AI服务层** | Apache Tika + 大模型API | 文档解析（Apache Tika提取文本）、技能提取与归一化（大模型）、匹配度计算（大模型） |
| **大模型服务** | 云端大模型API（如DeepSeek-V3.2）主用 / Ollama（Qwen2）降级 | 技能归一化、复杂语义理解、匹配报告生成 |
| **数据层** | PostgreSQL（结构化数据）、Neo4j 5.26.12（知识图谱）、Redis 8.6.2（缓存）、RocketMQ 5.5.0（消息队列）、RustFS（文件存储） | PostgreSQL存储用户/简历/职位/匹配记录；Neo4j存储技能图谱、人-岗关系；Redis缓存热点数据；RocketMQ处理解析任务队列；RustFS存储原始DOC/PDF文件 |
| **基础设施层** | Nginx（反向代理）、Spring Boot内置Tomcat（应用容器）、Windows/Linux（运行环境）、Docker（容器化） | 反向代理与静态资源服务、应用部署与运行、跨平台支持、可选容器化部署 |
