## 7.2 DDD各层职责说明

| 层级 | 包名 | 职责 |
|------|------|------|
| **领域层** | domain/ | 核心业务逻辑、领域模型、业务规则 |
| **应用层** | application/ | 用例编排、命令处理、查询处理 |
| **基础设施层** | infrastructure/ | 技术实现、持久化、第三方服务 |
| **接口层** | interface/ | HTTP接口、DTO定义、协议转换 |

## 7.3 核心模块说明

### 7.3.1 领域层（domain）

| 目录 | 说明 |
|------|------|
| model | 聚合根、实体：User、Resume、Job、MatchRecord等 |
| vo | 值对象：不可变对象，如SkillLevel枚举、MatchScore |
| event | 领域事件：如简历上传后发布ResumeUploadedEvent |
| service | 领域服务：跨实体的业务逻辑，如匹配规则计算 |
| repository | 仓库接口：仅定义接口，由基础设施层实现 |

### 7.3.2 应用层（application）

| 目录 | 说明 |
|------|------|
| service | 应用服务：编排领域服务，实现具体用例 |
| command | 命令对象：用于写操作（如注册、上传） |
| query | 查询对象：用于读操作（如列表查询） |

### 7.3.3 基础设施层（infrastructure）

| 目录 | 说明 |
|------|------|
| persistence/mapper | MyBatis-Plus Mapper接口 |
| persistence/repository | 仓库接口实现 |
| file | RustFS文件存储客户端 |
| mq | RocketMQ生产者和消费者 |
| graph | Neo4j图数据库客户端 |
| ai | AI服务客户端（DeepSeek、Ollama、Tika） |
| config | 各类配置类 |

### 7.3.4 接口层（interface）

| 目录 | 说明 |
|------|------|
| controller | RESTful API控制器 |
| converter | DTO与领域模型转换器 |
| dto/request | 请求参数DTO |
| dto/response | 响应数据DTO |

## 7.4 包依赖关系

```
┌─────────────────────────────────────────────────────┐
│                    interface层                       │
│  Controller → Converter → DTO/Request/Response      │
└─────────────────────┬─────────────────────────────────┘
                      │ 调用
                      ▼
┌─────────────────────────────────────────────────────┐
│                   application层                     │
│         AppService → Command/Query → Domain         │
└─────────────────────┬─────────────────────────────────┘
                      │ 调用
                      ▼
┌─────────────────────────────────────────────────────┐
│                     domain层                         │
│   Model ←→ Repository接口 ←→ DomainService         │
└─────────────────────┬─────────────────────────────────┘
                      │ 实现
                      ▼
┌─────────────────────────────────────────────────────┐
│                 infrastructure层                     │
│  Persistence ←→ MQ ←→ Graph ←→ AI ←→ File         │
└─────────────────────────────────────────────────────┘
```

## 7.5 命名规范

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 聚合根/实体 | 实体名 | User、Resume、Job |
| 值对象 | 描述性名词或枚举 | SkillLevel、MatchScore |
| 领域服务 | DomainService结尾 | MatchDomainService |
| 应用服务 | AppService结尾 | ResumeAppService |
| 仓库接口 | Repository结尾 | UserRepository |
| 仓库实现 | RepositoryImpl结尾 | UserRepositoryImpl |
| 命令对象 | Cmd结尾 | UploadResumeCmd |
| 查询对象 | Query结尾 | ResumeListQuery |
| 领域事件 | Event结尾 | ResumeParsedEvent |
| 控制器 | Controller结尾 | ResumeController |
| DTO请求 | Request结尾 | LoginRequest |
| DTO响应 | Response结尾 | ResumeDetailResponse |
