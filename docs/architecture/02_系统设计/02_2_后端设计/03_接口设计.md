# 接口设计

## 5.1 接口规范

- 协议：HTTPS
- 数据格式：JSON
- 认证方式：JWT Bearer Token
- 编码：UTF-8

## 5.2 认证接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /auth/sendCode | POST | 发送验证码（图片验证码校验） |
| /auth/register/person | POST | 个人用户注册 |
| /auth/register/company | POST | 企业用户注册 |
| /auth/login | POST | 用户登录 |
| /auth/logout | POST | 退出登录 |
| /auth/refreshToken | POST | 刷新Token |
| /auth/forgot-password | POST | 忘记密码-发送重置验证码 |
| /auth/reset-password | POST | 忘记密码-重置密码 |

## 5.3 个人用户接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /person/info | GET | 获取个人信息 |
| /person/info | PUT | 更新个人信息 |
| /person/resume/upload | POST | 上传简历 |
| /person/resume/list | GET | 简历列表 |
| /person/resume/{id} | DELETE | 删除简历 |
| /person/resume/{id}/default | PUT | 设置默认简历 |
| /person/resume/{id}/parse | POST | 重新解析简历 |
| /person/resume/{id}/detail | GET | 简历详情（解析后） |
| /person/graph | GET | 获取个人能力图谱 |
| /person/recommend/jobs | GET | 获取推荐职位列表 |
| /person/match/{jobId} | GET | 获取与职位的匹配详情 |

## 5.4 企业用户接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /company/info | GET | 获取企业信息 |
| /company/info | PUT | 更新企业信息 |
| /company/auth | POST | 提交认证资料 |
| /company/job | POST | 发布职位 |
| /company/job/list | GET | 职位列表 |
| /company/job/{id} | GET | 职位详情 |
| /company/job/{id} | PUT | 编辑职位 |
| /company/job/{id}/status | PUT | 上架/下架 |
| /company/job/{id}/parse | POST | 重新解析职位 |
| /company/job/{id}/graph | GET | 获取职位能力图谱 |
| /company/recommend/resumes | GET | 获取推荐候选人列表 |
| /company/match/{resumeId} | GET | 获取与候选人的匹配详情 |

## 5.5 管理员接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /admin/login | POST | 管理员登录 |
| /admin/user/list | GET | 用户列表 |
| /admin/user/{id}/status | PUT | 修改用户状态 |
| /admin/company/auth/list | GET | 待认证企业列表 |
| /admin/company/auth/{id} | PUT | 审核企业认证 |
| /admin/resume/list | GET | 简历库列表 |
| /admin/job/list | GET | 职位库列表 |
| /admin/skill/list | GET | 技能标签列表 |
| /admin/skill | POST | 添加技能标签 |
| /admin/skill/{id} | PUT | 编辑技能标签 |
| /admin/task/list | GET | 解析任务列表 |
| /admin/task/{id}/retry | POST | 重试任务 |
| /admin/statistics | GET | 系统统计面板 |
