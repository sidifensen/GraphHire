# GraphHire 图谱智聘 - 原型设计提示词（主要业务版）

---

## 一、项目定位

**GraphHire 图谱智聘** — 基于 AI 智能匹配与能力图谱的招聘平台。

**核心业务流程**：简历上传 → AI 解析技能 → 构建能力图谱 → 智能双向匹配 → 推荐通知

---

## 二、三端定位

| 端口 | 用户 | 设计风格 |
|------|------|----------|
| 用户端 | 求职者 | 现代简洁、蓝色系、卡片式布局 |
| 企业端 | 招聘方 | 同用户端设计体系、左侧菜单 |
| 管理端 | 管理员 | 灰色系、表格为主、功能导向 |

---

## 三、用户端

### 3.1 首页
- 顶部导航（Logo + 导航选项：首页/职位/公司/能力图谱 + 登录/注册 或 用户头像）
  - 导航选项（始终显示）：首页 / 职位 / 公司 / 能力图谱
  - 登录前显示：登录 / 注册按钮
  - 登录后显示：用户头像（点击展开下拉菜单：我的简历/我的投递/设置/退出登录）
- Hero 区域：大标题 + 搜索框（关键词/城市/薪资）+ 热门标签
- 推荐职位：卡片列表（公司名/薪资/地点/匹配度）
- 热门公司：Logo + 公司名 + 在招职位数
- 底部 Tab 导航（首页/职位/消息/我的）

### 3.2 登录页
- Tab 切换：密码登录 / 验证码登录
- 表单：账号 + 密码/验证码 + 登录按钮
- 底部链接：忘记密码 / 立即注册

### 3.3 注册页
- Tab 切换：我是求职者 / 我是招聘者
- 表单：手机号 + 验证码 + 密码 + 确认密码 + 协议勾选
- 招聘者注册额外字段：公司名称 + 统一社会信用代码

### 3.4 简历上传页
- 拖拽上传区域，支持 PDF/Word
- 上传进度条 + 文件信息展示
- 解析状态：解析中（loading）→ 成功（技能标签+置信度）/ 失败（重试）

### 3.5 简历管理页
- 简历列表：文件名/上传时间/解析状态/默认标识
- 操作：预览、删除、设为首要、重新解析
- 空状态引导上传

### 3.6 个人信息页
- 头像上传
- 基本信息：姓名、性别、年龄、手机、邮箱
- 教育信息：学校、学历、专业、毕业年份
- 求职意向：目标城市、期望薪资、岗位类型

### 3.7 能力图谱页
- 可视化画布：中心头像 + 辐射技能节点
- 技能节点可点击，弹出详情（技能名/掌握程度）
- 支持拖拽调整、缩放、全屏
- 可对比目标职位，高亮匹配技能

### 3.8 职位列表页
- 搜索筛选：关键词 + 城市 + 薪资范围 + 技能标签
- 职位卡片：Logo + 职位名 + 薪资 + 地点 + 技能标签 + 匹配度
- 排序：发布时间 / 匹配度 / 薪资
- 分页

### 3.9 匹配详情页
- 顶部：候选人 vs 职位信息对比
- 五维度雷达图：技能/经验/薪资/地点/学历
- AI 匹配原因文字说明
- 技能对比表：简历技能 vs 职位要求，匹配项高亮
- 按钮：投递职位 / 查看公司

### 3.10 通知中心
- Tab 分类：全部 / 简历解析 / 职位推荐 / 投递反馈 / 系统
- 通知列表：图标 + 标题 + 内容 + 时间 + 未读红点
- 操作：标为已读 / 删除

## 四、企业端

### 4.1 控制台首页
- 欢迎语 + 当天日期
- 待办卡片：待处理投递数 + 新匹配数
- 近期发布职位列表
- 快捷入口：发布新职位 / 查看推荐简历

### 4.2 登录/注册页
- 同用户端登录注册逻辑
- 企业注册需填写公司名称和统一社会信用代码

### 4.3 企业信息页
- 公司 Logo + 基本信息编辑（名称/联系人/联系方式/简介/官网）
- 营业执照上传
- 认证状态流转：
  - **待审核** → 管理员审批 → **已认证**（可发布职位）/ **已拒绝**（显示拒绝原因，可重新提交）

### 4.4 职位发布页
- 表单：职位名称 / 部门 / 人数 / 地点 / 薪资范围
- 技能要求：必填技能 + 优先技能（标签输入）
- 职位描述：富文本或文件上传
- 按钮：保存草稿 / 直接发布
- 注意：只有已认证企业才能发布职位

### 4.5 职位管理页
- Tab 状态：全部 / 草稿 / 已发布 / 已关闭
- 列表：职位名/状态/薪资/发布时间/投递数
- 操作：编辑 / 发布 / 下架 / 删除 / 查看图谱

### 4.6 推荐简历页
- 职位选择 + 匹配度筛选
- 简历卡片：头像/姓名/学历/城市/匹配度/技能标签
- 点击查看候选人详情

### 4.7 候选人详情页
- 候选人基本信息 + 简历预览
- 五维度匹配雷达图
- 技能对比表（匹配项高亮）
- AI 匹配原因说明
- 操作：查看更多简历 / 标记不合适

### 4.8 员工管理页
- 员工列表：账号/岗位角色（企业主/HR/招聘专员）/添加时间
- 添加员工：用户名 + 初始密码 + 岗位

### 4.9 通知中心
- Tab 分类：全部 / 候选人推荐 / 简历被查看 / 认证结果 / 系统
- 通知列表：图标 + 标题 + 内容 + 时间 + 未读红点
- 操作：标为已读 / 删除

## 五、管理端

### 5.1 仪表盘
- 大数字统计：用户数/企业数/简历数/职位数/匹配数
- 趋势折线图（近30天）
- 待办：待审核企业 + 失败任务数

### 5.2 企业审核页
- Tab 状态：全部 / 待审核 / 已通过 / 已拒绝
- 列表：企业名/信用代码/提交时间/认证材料/操作
- 操作：查看材料大图 / 通过 / 拒绝（填原因）

### 5.3 用户管理页
- 筛选：用户类型（个人/企业）/ 状态（正常/禁用/锁定）
- 列表：ID/用户名/类型/注册时间/状态/操作
- 操作：启用/禁用（选择临时或永久）

### 5.4 技能标签页
- 标签列表：名称/分类/同义词数/使用次数
- 添加/编辑标签（名称+分类+描述）
- 同义词管理（添加/删除）

### 5.5 任务监控页
- 统计卡片：待处理/处理中/成功/失败数
- 任务列表：ID/类型（简历解析/职位解析）/状态/重试次数/时间
- 失败任务高亮 + 重试按钮

## 六、核心交互

| 交互 | 说明 |
|------|------|
| 简历上传解析 | 拖拽上传 → 进度条 → AI解析中(loading) → 成功显示技能标签 / 失败可重试 |
| 能力图谱 | 中心头像向四周辐射技能节点，点击弹出详情，可与职位对比高亮匹配项 |
| 五维度匹配 | 雷达图展示技能/经验/薪资/地点/学历五个维度的匹配分数 |
| 投递流程 | 用户点击"投递" → 调用匹配接口 → 创建匹配记录 → 双方收到通知 |
| 认证流程 | 提交材料 → 待审核 → 已认证（可发布职位）/ 被拒绝（显示原因） |
| 通知系统 | 系统主动推送：简历解析完成/职位推荐/候选人推荐/认证结果/简历被查看 |

## 七、页面跳转

```
用户端：首页 ↔ 登录 ↔ 注册
       ↓
       简历上传 ↔ 简历管理
       ↓
       个人信息 / 能力图谱
       ↓
       职位列表 → 匹配详情 → 投递
       ↓
       通知中心

企业端：控制台 → 登录 ↔ 注册
       ↓
       企业信息（认证）
       ↓
       职位发布 ↔ 职位管理 → 候选人详情
       ↓
       推荐简历 → 候选人详情
       ↓
       员工管理 / 通知中心

管理端：仪表盘 → 登录
       ↓
       企业审核 / 用户管理 / 技能标签 / 任务监控
```
