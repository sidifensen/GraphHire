# GraphHire 图谱智聘 - 原型设计提示词（简略版）

---

## 一、项目定位

**GraphHire 图谱智聘** — 基于 AI 智能匹配与能力图谱的招聘平台。

**核心理念**：上传简历 → AI 解析技能 → 构建能力图谱 → 智能匹配推荐

---

## 二、三端定位

| 端口 | 用户 | 风格 |
|------|------|------|
| 用户端 | 求职者 | 现代简洁、蓝色系、卡片式 |
| 企业端 | 招聘方 | 同用户端设计体系 |
| 管理端 | 管理员 | 灰色系、功能导向、表格为主 |

---

## 三、用户端页面

| 页面 | 功能描述 |
|------|----------|
| **首页** | Hero区（标题+搜索框）+ 热门职位卡片 + 热门公司列表 |
| **登录/注册** | 切换Tab：密码登录 / 验证码登录；求职者 / 招聘者注册入口 |
| **简历上传** | 拖拽上传简历 + AI解析进度 + 解析结果展示（技能标签+置信度） |
| **简历管理** | 简历列表 + 状态标签 + 预览/删除/设默认操作 |
| **个人信息** | 头像 + 基本信息 + 教育背景 + 求职意向表单 |
| **能力图谱** | 可视化技能节点图（中心头像 + 辐射技能节点），支持点击查看详情 |
| **职位列表** | 搜索筛选 + 职位卡片列表（公司名/薪资/地点/匹配度） |
| **匹配详情** | 五维度雷达图 + 匹配原因 + 技能对比表 + 投递按钮 |
| **通知中心** | 通知列表 + 未读红点 + 分类Tab（匹配/投递/系统） |

---

## 四、企业端页面

| 页面 | 功能描述 |
|------|----------|
| **控制台** | 欢迎页 + 待办卡片（待处理投递/新匹配）+ 近期职位列表 + 快捷操作 |
| **登录/注册** | 账号密码登录；企业注册（含公司名+信用代码） |
| **企业信息** | 公司资料编辑 + 营业执照上传 + 认证状态流转（待认证→已认证） |
| **职位发布** | 表单：职位名/薪资/地点/技能要求/描述 + 保存草稿/直接发布 |
| **职位管理** | 职位列表 + 状态Tab（草稿/已发布/已关闭）+ 上下架/编辑/删除 |
| **推荐简历** | 系统推荐的简历卡片列表 + 匹配度展示 + 查看详情入口 |
| **候选人详情** | 候选人信息 + 五维度匹配雷达图 + 技能对比 + 面试邀请按钮 |
| **员工管理** | 员工列表 + 添加员工（用户名/密码/岗位） |
| **通知中心** | 同用户端通知列表 |

---

## 五、管理端页面

| 页面 | 功能描述 |
|------|----------|
| **仪表盘** | 大数字统计卡片（用户/企业/简历/职位/匹配）+ 趋势图 + 待办事项 |
| **企业审核** | 待审核企业列表 + 认证材料预览 + 通过/拒绝操作 |
| **用户管理** | 用户列表（个人/企业）+ 启用/禁用操作 |
| **技能标签** | 标签列表 + 添加/编辑标签 + 同义词管理 |
| **任务监控** | AI解析任务列表（待处理/成功/失败）+ 重试失败任务 |

---

## 六、核心交互

- **简历上传**：拖拽 → 进度条 → 解析中loading → 解析成功（技能标签）/ 失败（重试）
- **能力图谱**：中心头像 + 技能节点辐射 + 点击展开详情 + 对比职位高亮匹配项
- **匹配度展示**：五维度雷达图 + 百分比数字 + 匹配/不匹配技能高亮对比
- **认证流程**：提交材料 → 审核中 → 已认证（可发布职位）/ 被拒绝（显示原因）
