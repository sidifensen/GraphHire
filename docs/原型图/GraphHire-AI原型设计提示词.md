# GraphHire 图谱智聘 - 产品原型设计提示词

---

## 一、网站定位与愿景

**GraphHire 图谱智聘** 是一个基于 AI 智能匹配与能力图谱的招聘平台。

**核心理念**：让合适的人找到合适的岗位，让招聘更精准、更高效。

**核心价值链**：

```
个人用户：注册 → 完善简历 → AI解析技能 → 生成能力图谱 → 系统智能匹配 → 获得推荐职位 → 投递 → 收到面试邀请
企业用户：注册 → 提交认证 → 发布职位 → AI提取技能 → 生成职位图谱 → 系统智能推荐 → 筛选简历 → 发出面试邀请
平台管理员：审核企业 → 管理用户/内容 → 监控数据 → 保障平台健康运行
```

---

## 二、三端定位与角色

| 端口 | 用户角色 | 核心诉求 |
|------|----------|----------|
| **用户端** | 个人求职者 | 快速找到匹配的工作，展示自己的能力 |
| **企业端** | 招聘方 HR / 管理者 | 高效找到合适的候选人，管理招聘流程 |
| **管理端** | 平台运营 / 管理员 | 保障平台秩序，审核内容，管理用户 |

## 三、用户端（个人求职者）

### 3.1 首页

**功能描述**：面向求职者的主入口，展示平台核心价值和推荐内容

**包含模块**：

1. **导航栏**
   - 左侧：Logo + 平台名称"图谱智聘"
   - 右侧：登录 / 注册按钮（未登录时）
   - 右侧：消息图标 + 用户头像 + 下拉菜单（已登录时）
   - 下拉菜单：我的简历、我的收藏、账号设置、退出登录

2. **Hero 区域**
   - 主标题："AI 驱动，精准匹配，让每一次求职都更高效"
   - 副标题："上传简历，AI 智能解析你的能力图谱，自动匹配最适合你的职位"
   - 搜索框组件：
     - 关键词输入（职位名 / 技能）
     - 城市选择下拉
     - 薪资范围选择（可勾选"不限"）
     - 搜索按钮
   - 快捷标签：热门技能标签可点击

3. **推荐职位区**
   - 标题："为你推荐" + 显示数量标签
   - 职位卡片列表（横向滚动或网格）：
     - 每张卡片：职位名、公司 Logo、公司名、薪资、城市、匹配度百分比标签
     - 匹配度颜色规则：高(绿) / 中(黄) / 低(灰)
   - "查看更多" 按钮 → 职位列表页

4. **热门公司区**
   - 标题："热门雇主"
   - 公司卡片列表：
     - 公司 Logo + 公司名 + 行业标签 + 在招职位数
   - "查看全部公司" 按钮

5. **平台特色介绍区**（可选）
   - 图文介绍：能力图谱 / 智能匹配 / 精准推荐

6. **底部 Footer**
   - 链接：关于我们、帮助中心、联系方式
   - 版权信息

**包含模块**：

1. **Logo + 平台名称**

2. **登录方式 Tab 切换**

4. **验证码登录表单**
   - 手机号输入框
   - 验证码输入框 + 发送按钮（倒计时 60 秒）

**包含模块**：

   - 手机号输入框
   - 验证码输入框 + 发送按钮

### 3.5 简历上传页

**功能描述**：用户上传简历文件，系统自动解析技能

**包含模块**：

1. **上传区域**（核心）
   - 拖拽上传区：虚线边框 + 上传图标 + "将简历文件拖拽到此处，或点击上传"
   - 支持格式提示：PDF、Word
   - 文件大小限制提示：不超过 10MB

2. **上传进度**（上传时显示）
   - 进度条 + 百分比
   - 文件名显示

3. **上传完成状态**
   - 文件信息卡片：文件名、文件大小、上传时间
   - 解析状态标签：
     - "解析中" + 加载动画 + "AI 正在解析中，请稍候..."
     - "解析成功" + 绿色勾 + 显示置信度百分比
     - "解析失败" + 红色叉 + 失败原因 + "重新解析" 按钮

4. **解析结果展示**（解析成功后）
   - 提取的技能标签列表（可删除 / 添加）
   - 简历预览按钮 → 弹出简历预览

5. **操作按钮**
   - "保存简历" 按钮
   - "上传另一份" 按钮（可继续添加）

**功能描述**：用户查看和管理自己上传的所有简历

**包含模块**：

1. **页面标题**：我的简历 + 简历数量

2. **简历列表**
   - 每条记录：
     - 文件图标 + 文件名
     - 上传时间
     - 解析状态标签（成功 / 失败 / 待解析）
     - 匹配度（如果有）
     - 默认标识（星星图标，仅一份可设为默认）
     - 操作按钮组

3. **操作按钮组**
   - 预览（查看简历内容）
   - 设为默认（如果不是当前默认）
   - 重新解析（如解析失败）
   - 删除（确认弹窗）

4. **新增简历按钮**
   - 点击 → 跳转简历上传页

5. **空状态**
   - 无简历时显示：插图 + "还没有上传简历" + "上传简历，开启智能匹配" 按钮

### 3.7 个人信息页

**功能描述**：用户编辑和维护自己的基本信息

**包含模块**：

1. **头像上传区**
   - 圆形头像框 + 上传图标
   - 点击可更换头像
   - 默认头像（用户 initials）

2. **基本信息表单**
   - 姓名输入框
   - 性别选择（未知 / 男 / 女）
   - 年龄输入框（数字）

   - 邮箱输入框

3. **教育背景表单**
   - 学校名称输入框
   - 学历选择（高中 / 大专 / 本科 / 硕士 / 博士）
   - 专业输入框

4. **求职意向表单**
   - 目标城市选择（支持多选或单选）
   - 期望薪资范围选择（区间选择或输入框）
   - 岗位类型选择（全职 / 兼职 / 实习）

5. **保存按钮**

### 3.8 能力图谱页

**功能描述**：可视化展示用户的能力结构

**包含模块**：

1. **图谱画布**
   - 中心节点：用户头像 + 用户名
   - 辐射节点：各技能标签节点
   - 节点连线：中心到各技能节点
   - 节点大小：可根据匹配权重调整

2. **技能节点详情**（点击节点）
   - 弹出卡片显示：
     - 技能名称
     - 掌握程度（精通 / 熟悉 / 了解）
     - 相关经历说明（如有）
     - 与哪些职位匹配

3. **筛选控件**
   - 按技能分类筛选（下拉）
   - 按匹配度筛选（高 / 中 / 低）

4. **图谱操作**
   - 拖拽调整节点位置
   - 缩放控制（+ / -）
   - 全屏按钮
   - 重置视图按钮

5. **对比模式入口**
   - "对比目标职位" 按钮 → 选择职位后，图谱高亮匹配技能

### 3.9 职位搜索 / 列表页

**功能描述**：用户搜索和浏览所有在招职位

**包含模块**：

1. **搜索筛选区**
   - 关键词搜索框
   - 城市选择
   - 薪资范围选择
   - 技能标签选择（可多选）
   - 筛选按钮 / 重置按钮

2. **结果统计**
   - "共找到 X 个职位"

3. **职位列表**
   - 卡片式列表，每张卡片：
     - 公司 Logo + 公司名
     - 职位名称（加粗）
     - 薪资范围（醒目颜色）
     - 工作地点
     - 技能标签（最多显示 3 个，超出显示"+X"）
     - 匹配度百分比（如果有简历）
     - 发布时间（相对时间，如"3天前"）
   - 点击卡片 → 职位详情页

4. **排序控件**
   - 按发布时间排序
   - 按匹配度排序（如有简历）
   - 按薪资排序

5. **分页控件**
   - 页码 + 上下页
   - 每页条数选择（10 / 20 / 50）

### 3.10 职位详情页

**功能描述**：展示职位的完整信息

**包含模块**：

1. **职位基本信息卡片**

   - 薪资范围（醒目显示）
   - 工作地点
   - 招聘人数
   - 发布日期

2. **职位描述区**
   - Tab 切换：职位详情 / 任职要求 / 福利待遇
   - 富文本展示职位描述内容

3. **技能要求区**
   - 必填技能标签列表
   - 优先技能标签列表

4. **操作按钮组**
   - "立即投递" 按钮（如果已上传简历）
   - "收藏" 按钮
   - "查看匹配详情" 按钮（如有简历）

5. **公司信息区**
   - 公司名称 + Logo
   - 公司简介（截断 + "展开"）
   - 官网链接

### 3.11 匹配详情页

**功能描述**：展示用户与职位的匹配分析结果

**包含模块**：

1. **对比双方信息**
   - 左侧：用户头像 + 姓名 + 主要技能标签
   - 中间：匹配度大数字（百分比）+ 匹配等级（高/中/低）
   - 右侧：职位名称 + 公司名 + 薪资

2. **五维度匹配雷达图**
   - 技能匹配
   - 经验匹配
   - 薪资匹配
   - 地点匹配
   - 学历匹配
   - 各维度分数（0-100）

3. **匹配原因说明**
   - AI 生成的文字描述：
   - "您的 Java 技能（精通）与职位要求高度匹配"
   - "您的期望薪资略高于职位薪资范围"
   - "您的目标城市与职位地点完全匹配"

4. **技能对比详情**
   - 表格或双列对比：
     - 左列：简历技能（用户拥有）
     - 右列：职位要求（岗位需要）
     - 匹配项用绿色高亮
     - 不匹配项用灰色

5. **操作按钮**
   - "投递此职位" 按钮
   - "收藏此职位" 按钮
   - "返回" 按钮

**功能描述**：系统根据用户简历智能推荐的职位列表

**包含模块**：

1. **页面标题**：为你推荐 + 推荐数量

2. **匹配度说明**
   - 简短说明："基于你的简历技能和求职意向为你推荐"

3. **推荐职位列表**
   - 同职位列表页卡片样式，但匹配度必须显示
   - 按匹配度从高到低排序

4. **筛选与排序**
   - 匹配度范围筛选（80%以上 / 60%以上 / 全部）
   - 排序（匹配度 / 发布时间 / 薪资）

5. **空状态**
   - "暂无推荐职位" + "完善简历可以获得更精准的推荐"

## 八、设计规范

### 8.1 响应式设计规范

**设计目标**：适配电脑端（桌面端）与手机端（移动端），确保两端的用户体验一致且流畅。

#### 8.1.1 断点定义

| 设备类型 | 屏幕宽度范围 | 设计策略 |
|----------|--------------|----------|
| **手机端（Mobile）** | < 768px | 单列布局，触控优先 |
| **平板端（Tablet）** | 768px - 1024px | 双列布局，混合交互 |
| **电脑端（Desktop）** | > 1024px | 多列布局，悬停交互 |

#### 8.1.2 手机端适配要求

1. **布局转换**
   - 多列卡片布局 → 单列纵向滚动
   - 侧边导航 → 底部 Tab 导航（用户端）或汉堡菜单（企业端/管理端）
   - 表格 → 卡片列表

2. **触控优化**
   - 按钮最小点击区域：44px × 44px
   - 列表项内边距：≥ 12px
   - 输入框高度：≥ 44px
   - 支持手势操作：下拉刷新、上拉加载更多

3. **内容适配**
   - 隐藏次要信息，优先展示核心内容
   - 技能标签在移动端最多显示 2 个，其余折叠
   - 职位列表卡片：Logo 缩小，信息紧凑排列

4. **导航适配**
   - 用户端：底部固定 Tab 栏（首页、职位、消息、我的）
   - 企业端/管理端：顶部导航栏 + 侧滑抽屉菜单

#### 8.1.3 电脑端适配要求

1. **布局策略**
   - 最大内容宽度：1280px（居中显示）
   - 导航栏高度：64px
   - 侧边栏宽度：240px（可折叠至 64px）

2. **交互增强**
   - 悬停态显示次要操作按钮
   - 表格支持列排序
   - 列表支持批量选择操作
   - 键盘快捷键支持

#### 8.1.4 响应式组件原则

| 组件类型 | 手机端 | 电脑端 |
|----------|--------|--------|
| 搜索框 | 整宽显示 | 与筛选条件并排 |
| 筛选器 | 折叠为"筛选"按钮，点击展开面板 | 侧边筛选面板常驻 |
