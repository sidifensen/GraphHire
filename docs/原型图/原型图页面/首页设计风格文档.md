## 1. 设计风格定义

**风格名称：** Modern Minimal SaaS / Soft UI

**适用场景：** AI 产品、SaaS 平台、科技创业公司官网

**参考案例：** Vercel、Linear、Notion、Raycast 等现代科技产品

## 2. 视觉特征

### 2.1 玻璃拟态（Glassmorphism）

```
.glass-card {
    background: rgba(255, 255, 255, 0.7);
    backdrop-filter: blur(20px);
    -webkit-backdrop-filter: blur(20px);
}
```

**应用场景：**
- 搜索框容器
- 模态框背景
- 悬浮工具栏

### 2.2 微分层（Micro-elevation）

```html
<!-- 卡片悬浮效果 -->
<div class="transition-all hover:-translate-y-2 hover:shadow-2xl hover:shadow-primary/5">
```

**特点：**
- 悬浮高度：-8px (`-translate-y-2`)
- 阴影加深，配合 hover 状态
- 过渡时间：300ms ease

### 2.3 渐变光晕装饰

```html
<!-- Hero 区域背景光晕 -->
<div class="absolute top-[-10%] left-[-10%] w-[50%] h-[80%] bg-primary-fixed opacity-20 blur-[120px] rounded-full"></div>
<div class="absolute bottom-[-10%] right-[-10%] w-[40%] h-[60%] bg-secondary-fixed opacity-20 blur-[100px] rounded-full"></div>
```

**配色：**
- 左上光晕：主色蓝 (primary-fixed)
- 右下光晕：青色 (secondary-fixed)

## 4. 字体系统

### 4.1 字体族

```html
<link href="https://fonts.googleapis.com/css2?family=Manrope:wght@200..800&family=Inter:wght@300..700&display=swap" rel="stylesheet"/>
```

| 用途 | 字体 | 字重 |
|------|------|------|
| 标题 (Headline) | Manrope | 700, 800 |
| 正文 (Body) | Inter | 300, 400, 500, 600, 700 |
| 标签 (Label) | Inter | 400, 500, 600 |

### 4.2 字号规范

| 场景 | 字号 | 行高 |
|------|------|------|
| Hero 标题 | 3.75rem (60px) / 4.5rem (72px) | tight (1.1) |
| 区块标题 | 2.25rem (36px) | tight (1.2) |
| 卡片标题 | 1.25rem (20px) | normal |
| 正文 | 1rem (16px) | relaxed (1.625) |
| 标签文字 | 0.75rem (12px) | normal |

## 5. 圆角规范

```javascript
borderRadius: {
    "DEFAULT": "0.25rem",   // 4px - 小按钮、标签
    "lg": "0.5rem",         // 8px - 输入框
    "xl": "0.75rem",        // 12px - 卡片
    "full": "9999px"        // 圆形头像、胶囊按钮
}
```

## 8. 动效规范

### 8.1 过渡时长

```html
<!-- 标准过渡 -->
transition-all

<!-- 按钮悬浮缩放 -->
hover:scale-105
```

### 8.2 动画类型

| 元素 | 动效 | 时长 |
|------|------|------|
| 卡片悬浮 | translateY(-8px) + 阴影加深 | 300ms |
| 按钮悬浮 | scale(1.05) | 200ms |
| 图标悬停 | 颜色变化 | 200ms |
| 雇主Logo | grayscale → color + opacity | 300ms |

## 9. 组件清单

### 9.1 导航栏 (TopNavBar)

```html
<nav class="fixed top-0 w-full z-50 bg-white/70 backdrop-blur-md">
    <!-- Logo + 导航链接 + 用户头像 -->
</nav>
```

**特点：**
- 固定顶部
- 玻璃效果背景
- 高度：64px (`h-16`)

### 9.2 搜索框 (Search Bar)

```html
<div class="glass-card p-2 rounded-2xl border border-outline-variant/20">
    <!-- 输入框 + 搜索按钮 -->
</div>
```

### 9.3 职位卡片 (Job Card)

```html
<div class="bg-surface-container-lowest p-8 rounded-xl hover:-translate-y-2 hover:shadow-2xl">
    <!-- 公司Logo + 匹配度标签 + 职位信息 + 薪资 + 申请按钮 -->
</div>
```

### 9.4 技能标签 (Skill Chip)

```html

    <!-- 技能名称 -->
</span>
```

### 9.5 雇主Logo网格

```html
<div class="grayscale opacity-60 hover:grayscale-0 hover:opacity-100">
    <!-- 公司名称文字 Logo -->
</div>
```

## 13. 实施建议

### 13.1 新页面开发

1. 引入 Tailwind CDN
2. 复制 `tailwind.config` 到项目
3. 引入 Google Fonts 和 Material Icons
4. 按照上方配色、字体规范使用

### 13.2 组件复用

建议封装以下组件：
- `<GlassCard>` - 玻璃效果卡片
- `<JobCard>` - 职位卡片
- `<SkillChip>` - 技能标签
- `<TopNavBar>` - 顶部导航
- `<SearchBar>` - 搜索栏

### 13.3 注意事项

- 渐变光晕元素需使用 `position: absolute` 定位
- 背景模糊效果需加 `-webkit-` 前缀兼容 Safari
- 卡片 hover 效果需设置 `transition-all`
- 深色模式通过 `class="dark"` 触发（已在 HTML 中预留）
