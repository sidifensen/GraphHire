## 第一部分：用户端提示词

请你为我生成 **GraphHire 图谱智聘的用户端原型**，这是一个基于 AI 智能匹配与能力图谱的招聘平台。

这一次你只生成 **用户端**，不要生成企业端，不要生成管理端。

用户端整体风格要求：

- 所有页面必须是同一套设计系统
- 所有页面必须统一使用玻璃拟态（Glassmorphism）风格
- 首页、登录页、注册页、简历相关页面、职位页面、通知页面必须看起来属于同一个产品
- 按钮、卡片、输入框、标签、状态色、导航栏样式必须统一
- 页面文案必须使用中文
- 所有页面界面必须是中文，不允许生成英文界面
- 导航、标题、按钮、表单、表格、提示语、状态文案、弹窗文案必须全部使用中文
- 禁止在提示词中出现任何颜色代码（如 #xxx、rgb、十六进制等），所有颜色由 AI 根据玻璃拟态风格自行搭配

请按下面顺序生成用户端页面：

1. 首页（现代化落地页）
2. 登录页
3. 注册页
4. 简历上传页
5. 简历管理页
6. 个人信息页
7. 能力图谱页
8. 职位列表页
9. 匹配详情页
10. 通知中心

每一页都要有真实模块，不要只写标题。

用户端页面要求如下：

### 1. 首页（现代化落地页 - 重制版）

**整体风格要求（强制）：**
- 主题风格：Glassmorphism（玻璃拟态）+ 粒子动效 + 微交互设计
- 毛玻璃卡片 + 流光效果 + 粒子流动背景 + 深色沉浸式底色
- 整体呈现高端科技感、现代感、未来感
- 所有颜色由 AI 根据玻璃拟态风格自行搭配，不要写死任何颜色值
- 禁止在提示词中出现任何颜色代码，禁止写死任何色值

**页面结构：**

**① 顶部导航栏**
- 固定顶部导航栏，毛玻璃半透明效果
- Logo（左侧）+ 导航项（首页、职位、公司、能力图谱）+ 右侧登录/注册按钮
- 滚动时背景从完全透明 → 半透明加深，实现视觉层次过渡
- 导航项悬停：文字发光 + 下划线从左向右滑入
- 移动端：汉堡菜单触发侧边抽屉

**② Hero 区域（核心视觉区）**

- 主标题：超大字号 + 渐变文字效果（使用 CSS 渐变让文字有光泽感）
- 副标题：简洁有力的价值主张文案
- AI 匹配示意动画区：
  - 两个节点（代表求职者与职位）用动态连线连接
  - 连线生成时有粒子散射效果
  - 节点周围有呼吸光晕
  - 持续循环动画吸引注意力
- 主 CTA 按钮组：「立即开始」+ 「查看职位」，按钮有脉冲光晕动画
- 背景：粒子流动（多个粒子以不同速度、方向移动）+ 渐变光晕
- 动画流程：页面加载 → 粒子淡入背景 → Hero 文字打字机效果逐字显示 → CTA 按钮入场带光晕脉冲

**③ 搜索区块**
- 毛玻璃大卡片包裹搜索栏
- 三栏布局：关键词输入框 | 城市选择下拉 | 薪资范围选择
- 搜索按钮有流光边框动效
- 热门搜索标签胶囊横向排列，点击即填入搜索框
- 动画：搜索区块入场时从下向上依次浮入（搜索框 → 标签）

**④ 核心能力展示区（三列卡片）**

- 卡片标题：智能简历解析 / 能力图谱构建 / AI 精准匹配
- 每张卡片包含：图标（带微动画）+ 标题 + 2-3行简短描述
- 卡片样式：毛玻璃背景 + 细边框 + 内阴影
- 悬停效果：
  - 卡片整体上浮（translateY: -8px）
  - 阴影扩散增大
  - 边框出现流光
  - 内部图标有轻微放大 + 颜色变化
- 进入视口动画：卡片依次淡入上移，间隔 150ms

**⑤ 数据统计区**
- 横向排列 4 个关键数据指标：已解析简历数 | 入驻企业数 | 成功匹配数 | 用户满意度
- 每个数据块：大数字（带计数动画）+ 单位 + 描述文字
- 数字从 0 滚动到目标值动画
- 背景有微弱的渐变光效

**⑥ 推荐职位轮播区**
- 标题：「精选推荐职位」+ 「查看更多」链接
- 横向滚动职位卡片列表，支持拖拽滑动
- 每张职位卡片包含：
  - 职位名称 + 公司名称 + 工作地点
  - 薪资范围（高亮显示）
  - 匹配度进度条（用颜色条表示匹配度百分比）
  - 技能标签（最多显示3个）
  - 发布时间
- 卡片悬停：整体微上浮 + 边框发光
- 动画：无限循环滚动，进入视口时淡入上移

**⑦ 能力图谱展示区**
- 标题：「你的能力图谱」+ 引导语
- 缩略版能力图谱可视化展示（动态圆环图 + 技能节点）
- 右侧：图谱统计概览（技能数量、主要领域、综合评分）
- CTA 按钮：「生成我的图谱」
- 背景有流动的粒子连线效果，模拟神经网络

**⑧ 用户评价 / 成功案例区**
- 标题：「他们找到了理想工作」
- 卡片轮播：用户头像 + 姓名 + 职位 + 简短评价
- 卡片有毛玻璃效果，悬停时轻微放大

**⑨ 底部 CTA 区**
- 大面积深色渐变背景（与页面形成对比）
- 主标题 + 描述文案
- 双按钮：上传简历 / 浏览职位
- 背景粒子加速向上流动
- 按钮悬停：边框流光 + 背景色渐变

**⑩ Footer**
- 简洁链接导航（关于我们、帮助中心、联系我们、用户协议）
- 版权信息
- 毛玻璃分隔线

**动效完整清单（必须全部实现）：**

| 位置 | 动效要求 |
|------|----------|
| 页面加载 | 粒子背景淡入（500ms）→ Hero 文字打字机效果（逐字显示，50ms/字）→ CTA 按钮脉冲光晕（无限循环） |
| 导航栏 | 悬停时文字发光 + 下划线从左向右滑入（300ms ease-out） |
| 导航滚动 | 背景从完全透明 → 半透明加深（滚动超过 50px 触发） |
| 搜索框 | focus 时边框发光（box-shadow 扩散）+ 轻微放大（scale: 1.02） |
| 能力卡片 | 悬停上浮（translateY: -8px）+ 阴影扩散 + 边框流光 + 图标放大 |
| 数据统计 | 数字计数滚动动画（0 → 目标值，1.5s ease-out） |
| 职位卡片 | 进入视口时淡入上移（opacity: 0→1, translateY: 20px→0，400ms） |
| 能力图谱区 | 背景粒子连线持续流动动画 |
| 底部 CTA | 粒子速度加快（背景粒子加速向上）+ 按钮悬停流光 |
| 全局 | 平滑滚动、过渡动画统一使用 cubic-bezier(0.4, 0, 0.2, 1) |

**禁止出现的元素：**
- 任何颜色代码（#xxx、rgb、hex）
- 写死的色值定义
- 纯平面无动效的静态设计
- 平庸的普通设计

### 2. 登录页

- 密码登录 / 验证码登录切换标签
- 手机号输入框
- 密码或验证码输入框
- 登录按钮
- 忘记密码链接
- 去注册入口链接
- 错误提示（输入框下方红色提示）
- 登录中状态（按钮显示加载动画）

### 3. 注册页

- 求职者 / 招聘者身份切换
- 手机号、验证码、密码、确认密码输入框
- 协议勾选复选框（默认不勾选）
- 招聘者额外字段：公司名称、统一社会信用代码
- 注册按钮
- 去登录入口

### 4. 简历上传页

- 拖拽上传区（大面积虚线框，拖拽时边框变色 + 背景变色）
- 支持 PDF / Word 提示文字
- 上传进度条（动态进度动画）
- 文件信息展示（文件名、文件大小、上传时间）
- 简历解析状态：排队中 → 解析中（进度动画）→ 解析成功 / 解析失败
- 解析结果摘要卡片
- 失败重试按钮

### 5. 简历管理页

- 简历列表（表格或卡片列表）
- 每条简历显示：文件名、上传时间、解析状态（标签）、是否默认简历标记
- 操作按钮：预览、删除、设为首要、重新解析
- 空状态：引导上传提示 + 上传按钮
- 分页或无限滚动

### 6. 个人信息页

- 头像上传区（点击触发选择，支持拖拽）
- 基本信息编辑区：姓名、性别、出生年月、联系电话、邮箱
- 教育信息编辑区：学校、学历、专业、毕业时间
- 求职意向编辑区：期望职位、期望城市、期望薪资范围
- 保存按钮
- 编辑与保存状态切换

### 7. 能力图谱页

- 能力图谱可视化区域（SVG 或 Canvas 绘制）
- 中心人物节点（头像或图标）
- 技能节点围绕中心分布（不同大小表示熟练度）
- 点击技能节点弹出详情气泡（技能名称、熟练度、来源）
- 图谱统计信息侧边栏：总技能数、主要领域分布、综合评分
- 对比目标职位模式：叠加显示目标职位技能要求
- 底部操作栏：缩放控制（+ / -）、重置、拖拽模式切换、全屏按钮

### 8. 职位列表页

- 顶部搜索栏（关键词 + 筛选按钮）
- 筛选区：城市（多选）、薪资范围（滑块或区间选择）、技能标签（多选）、排序（最新发布/最高薪资/最高匹配度）
- 职位卡片列表
- 每张卡片：职位名、公司名、公司Logo、地点、薪资范围、经验要求、技能标签、匹配度百分比条
- 分页或无限滚动加载
- 筛选条件标签（可删除单个）

### 9. 匹配详情页

- 顶部：候选人信息卡片 + 目标职位信息卡片并排
- 总匹配度：大数字 + 进度环动画
- 五维度匹配分析（技能、经验、学历、地点、薪资），每个维度有雷达图或进度条
- AI 匹配原因说明（自然语言描述）
- 技能对比明细表：技能名称 | 候选人掌握程度 | 职位要求程度
- 底部操作栏：投递简历按钮、查看公司详情按钮、收藏按钮

### 10. 通知中心

- 顶部 Tab 栏：全部 / 简历解析 / 职位推荐 / 投递反馈 / 系统
- 通知列表，每条通知包含：图标（不同类型不同图标）、标题、摘要、时间、未读标记
- 滑动操作：标记已读、删除
- 空状态提示
- 全部已读按钮

用户端必须体现这些流程：

- 简历上传解析流程
- 能力图谱构建与职位对比流程
- 智能匹配与投递流程
- 通知触达流程

## 第二部分：企业端提示词

请你为我生成 **GraphHire 图谱智聘的企业端原型**，这是一个基于 AI 智能匹配与能力图谱的招聘平台。

这一次你只生成 **企业端**，不要生成用户端，不要生成管理端。

企业端整体风格要求：

- 所有页面必须是同一套设计系统
- 所有页面必须统一使用玻璃拟态风格
- 控制台、认证、职位、候选人、员工、通知页面必须看起来属于同一个产品
- 按钮、卡片、表单、表格、标签、状态色、导航样式必须统一
- 页面文案必须使用中文
- 所有页面界面必须是中文，不允许生成英文界面
- 导航、标题、按钮、表单、表格、提示语、状态文案、弹窗文案必须全部使用中文
- 禁止在提示词中出现任何颜色代码，禁止写死任何色值

请按下面顺序生成企业端页面：

1. 企业控制台首页
2. 企业登录 / 注册页
3. 企业信息与认证页
4. 职位发布页
5. 职位管理页
6. 推荐简历页
7. 候选人详情页
8. 员工管理页
9. 企业通知中心

每一页都要有真实模块，不要只写标题。

企业端页面要求如下：

### 1. 企业控制台首页

- 欢迎语（大标题 + 当前日期）
- 四个核心数据卡片：新投递数、新匹配候选人数、在招职位数、本周浏览数
- 每个卡片有图标 + 数字 + 描述 + 趋势指示（上升/下降箭头）
- 近期发布职位列表（最新5条）
- 快捷操作入口卡片：发布新职位、查看推荐简历、处理新投递
- 待办事项提醒区域

### 2. 企业登录 / 注册页

- 企业身份标识明确（企业入口提示）
- 手机号 + 密码登录 / 手机号 + 验证码登录切换
- 企业注册表单：公司名称、统一社会信用代码、联系人、密码、确认密码、验证码
- 登录 / 注册切换标签
- 错误提示和状态反馈（输入框下方提示）
- 协议勾选（注册时）

### 3. 企业信息与认证页

- 企业 Logo 上传区
- 公司基础信息：名称、简称、规模、人数、行业
- 详细信息：联系人、联系电话、公司官网、公司简介
- 营业执照上传区（拖拽上传，支持预览）
- 认证状态标签：待审核（黄） / 已认证（绿） / 已拒绝（红）
- 认证拒绝原因展示（红色提示框）
- 重新提交认证入口（拒绝后显示）

### 4. 职位发布页

- 职位基本信息：职位名称、部门、招聘人数、工作地点
- 薪资范围：月薪 / 年薪切换，最低薪资 ~ 最高薪资
- 技能要求：必填技能（可多选/搜索添加）+ 加分技能（可多选/搜索添加）
- 职位描述：富文本编辑器（支持加粗、列表等基础格式）
- 任职要求：富文本编辑器
- 工作福利标签选择
- 保存草稿按钮、预览按钮、发布按钮
- 未认证企业的提示（需要先完成企业认证才能发布职位）

### 5. 职位管理页

- 顶部 Tab：全部 / 草稿 / 已发布 / 已关闭
- 筛选：发布时间范围、职位类型
- 职位列表（表格形式）
- 每行显示：职位名称、发布状态、收到投递数、浏览数、发布时间、操作按钮
- 操作：编辑、发布/下架、删除、查看图谱
- 批量操作：批量下架、批量删除
- 分页

### 6. 推荐简历页

- 顶部职位筛选下拉（选择要查看的职位）
- 匹配度区间筛选滑块
- 简历卡片列表
- 每张卡片：求职者头像、姓名、学历、城市、期望薪资、主要技能标签（最多4个）、匹配度百分比
- 卡片底部操作：查看详情、不合适（标记后隐藏）
- 已标记不合适列表（可撤销）

### 7. 候选人详情页

- 左侧：候选人基本信息卡（姓名、头像、学历、工作年限、期望职位、期望薪资、联系电话）
- 右侧：与目标职位的对比分析区
- 五维度匹配雷达图：技能匹配度、经验匹配度、学历匹配度、地点匹配度、薪资匹配度
- 技能匹配明细：职位要求技能 vs 候选人掌握技能，逐一对比
- AI 推荐理由（自然语言描述）
- 候选人简历预览区（PDF 预览或文本展示）
- 底部操作栏：邀请面试、收藏简历、标记不合适、下载简历

### 8. 员工管理页

- 员工列表（表格形式）
- 每行：姓名、邮箱、手机、角色（企业主/管理员/招聘专员）、状态（正常/禁用）、最后登录时间、操作
- 添加员工弹窗表单：姓名、邮箱、手机、角色选择、发送邀请
- 编辑员工角色弹窗
- 员工角色说明：企业主（最高权限）、管理员（管理权限）、招聘专员（查看和筛选）
- 禁用/启用员工操作

### 9. 企业通知中心

- 顶部 Tab：全部 / 候选人推荐 / 简历被查看 / 投递通知 / 认证结果 / 系统
- 通知列表，每条：图标、标题、摘要、时间、已读/未读标记
- 滑动操作：标记已读、删除
- 全部已读按钮
- 空状态

企业端必须体现这些流程：

- 企业认证流程
- 职位发布与职位管理流程
- 推荐候选人与候选人筛选流程

## 第三部分：管理端提示词

请你为我生成 **GraphHire 图谱智聘的管理端原型**，这是一个基于 AI 智能匹配与能力图谱的招聘平台。

- 禁止在提示词中出现任何颜色代码，禁止写死任何色值

请按下面顺序生成管理端页面：

1. 管理端仪表盘

管理端页面要求如下：

### 1. 管理端仪表盘

- 顶部：平台名称 + 管理员信息 + 退出登录
- 核心数据卡片（4-6个）：总用户数、总企业数、总简历数、总职位数、本月匹配成功数、本月活跃用户数
- 每个卡片：图标 + 数字 + 描述文字 + 环比增长率
- 近30天趋势折线图（可切换指标：新增用户、新增企业、新增简历、新增职位、匹配数）
- 待办事项区域：待审核企业（数字 + 快速处理入口）、失败任务（数字 + 查看详情入口）
- 最近活动日志（最近10条操作记录）

### 2. 企业审核页

- 顶部 Tab：全部 / 待审核 / 已通过 / 已拒绝
- 筛选：提交时间范围
- 企业列表（表格形式）
- 每行：企业名称、联系人、联系电话、提交时间、认证状态、操作
- 操作按钮：查看详情（弹出侧边抽屉展示认证材料）、通过、拒绝
- 拒绝操作：点击拒绝 → 弹出模态框要求填写拒绝原因 → 确认提交
- 批量操作：批量通过、批量拒绝
- 分页

### 3. 用户管理页

- 顶部 Tab：全部 / 求职者 / 企业用户
- 状态筛选：正常 / 禁用 / 锁定
- 搜索框：按用户名、手机、邮箱搜索
- 用户列表（表格形式）
- 每行：头像、用户名、手机、邮箱、用户类型、状态、注册时间、最后登录、操作
- 操作：查看详情、禁用/启用、解锁（锁定状态时显示）
- 禁用操作：弹出确认框说明后果（将禁止该用户登录）
- 分页

### 4. 技能标签管理页

- 左侧：标签分类树（IT技术、金融、运营、设计、管理等）
- 右侧：选中分类下的标签列表
- 标签列表：标签名、分类、同义词数量、使用次数、操作
- 操作：编辑（弹出模态框修改标签名和分类）、管理同义词、删除
- 添加标签按钮 → 弹出模态框：输入标签名 + 选择分类
- 同义词管理：点击管理 → 侧边抽屉展示该标签的所有同义词列表 + 添加同义词输入框 + 删除同义词按钮
- 标签使用次数统计（数字形式）

### 5. 任务监控页

- 顶部统计卡片：待处理任务数、处理中任务数、成功率、失败数
- 任务类型筛选：全部 / 简历解析 / 职位解析 / 匹配计算
- 任务状态筛选：全部 / 排队中 / 处理中 / 成功 / 失败
- 任务列表（表格形式）
- 每行：任务ID、任务类型、任务状态、创建时间、开始处理时间、结束时间、重试次数、操作
- 成功任务显示绿色状态标签
- 失败任务：红色状态标签 + 高亮背景
- 操作：查看详情（弹出模态框展示任务执行日志）、重试（失败任务可用）
- 批量重试：选择多条失败任务 → 批量重试按钮
- 分页

管理端必须体现这些流程：

- 企业审核流程
- 用户管理流程（启用/禁用/解锁）
- 技能标签治理流程
