# GraphHire 图谱智聘 - AI 原型设计提示词

> 本文件用于把原型提示词拆成三个独立部分。
> 使用方式：不要一次性全部投喂。请按需复制下面某一个部分，单独投喂给原型生成网站。

---

## 第一部分：用户端

请你生成 **GraphHire 图谱智聘的用户端原型**，基于 AI 智能匹配与能力图谱的招聘平台。

只生成用户端

**风格要求：** 现代化设计，动效丰富，视觉精致，整体属于同一套设计系统。

**页面文案全部使用中文。**

### 页面清单

1. 首页（落地页）
2. 登录页
3. 注册页
4. 简历上传页
5. 简历管理页
6. 个人信息页
7. 能力图谱页
8. 职位列表页
9. 匹配详情页
10. 通知中心

### 业务流程

- 简历上传 → AI 解析 → 能力图谱构建
- 能力图谱与目标职位对比 → 智能匹配
- 匹配结果查看 → 投递简历
- 系统通知触达

## 第二部分：企业端

请你生成 **GraphHire 图谱智聘的企业端原型**，基于 AI 智能匹配与能力图谱的招聘平台。只生成企业端，不要生成用户端和管理端。

**风格要求：** 现代化设计，动效丰富，视觉精致，整体属于同一套设计系统。

**页面文案全部使用中文。**

### 页面清单

1. 企业控制台首页
2. 企业登录 / 注册页
3. 企业信息与认证页
4. 职位发布页
5. 职位管理页
6. 推荐简历页
7. 候选人详情页
8. 员工管理页
9. 企业通知中心

### 业务流程

- 企业注册 → 认证审核
- 认证通过 → 发布职位
- 发布职位 → 查看推荐候选人 → 筛选 → 邀请面试
- 系统通知触达

## 第三部分：管理端

请你生成 **GraphHire 图谱智聘的管理端原型**，基于 AI 智能匹配与能力图谱的招聘平台。只生成管理端，不要生成用户端和企业端。

**风格要求：** 现代化设计，动效丰富，视觉精致，整体属于同一套设计系统。

**页面文案全部使用中文。**

### 页面清单

1. 管理端仪表盘
2. 企业审核页
3. 用户管理页
4. 技能标签管理页
5. 任务监控页

### 业务流程

- 企业认证审核（通过 / 拒绝）
- 用户管理（启用 / 禁用 / 解锁）
- 技能标签治理（增删改、分类、同义词管理）
- 任务监控与失败重试
