# 聊天页面双栏重设计验收标准

- [ ] 用户端 `/chat` 桌面端展示左会话列表与右聊天面板同屏布局。
- [ ] 企业端 `/enterprise/chat` 桌面端展示左会话列表与右聊天面板同屏布局。
- [ ] 移动端保持“会话列表 -> 聊天详情”单栏路径，用户端与企业端均可进入详情聊天。
- [ ] 聊天头部展示岗位负责人、公司、岗位、薪资、地点，并提供“查看职位”入口。
- [ ] 消息气泡中双方消息均显示头像（真实头像或占位头像）。
- [ ] 消息时间按天显示分隔，同一天内不重复逐条显示时间。
- [ ] 输入区包含表情按钮，点击后可打开面板并插入表情。
- [ ] 输入区包含图片按钮，选择图片后可触发发送流程。
- [ ] 输入区包含“发送简历”按钮，用户端点击后可直接发送默认简历（第一份简历），无需选择器。
- [ ] 企业端保留原有发送面试通知能力，不影响文本与图片消息发送。
- [ ] 改造后通过前端构建与测试命令验证。
