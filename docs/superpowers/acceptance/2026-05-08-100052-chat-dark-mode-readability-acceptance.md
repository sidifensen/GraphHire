# 聊天页暗色可读性修复验收标准

- [ ] 会话搜索输入框在暗色下不出现白底，且使用主题化背景/边框类。
- [ ] 底部表情按钮与图片按钮在暗色下不出现白底，且使用主题化背景/边框类。
- [ ] 文本输入框在暗色下不出现白底，且使用主题化背景/边框类。
- [ ] 消息滚动容器使用 `chat-scrollbar`，滚动条轨道与滑块颜色来源于主题变量。
- [ ] 对方消息气泡在暗色下不出现白底，文本与容器使用主题 token 类。
- [ ] 相关回归测试通过，且断言不存在硬编码 `bg-white`。
- [ ] 前端全量验证通过：`npm run test:run`、`npm run build`。
