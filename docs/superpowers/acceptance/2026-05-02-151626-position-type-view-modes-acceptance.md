# 职位类型管理新增展示模式 验收标准

- [ ] 在 `职位类型管理` 页签可看到四个模式切换按钮：`树+详情`、`树形表格`、`路径面包屑视图`、`分栏级联视图`。
- [ ] 切换到 `路径面包屑视图` 时：
  - [ ] 显示当前选中节点面包屑路径（示例：`技术 / 后端开发 / Java`）。
  - [ ] 左侧仅显示当前节点同级节点，不展示整棵树。
  - [ ] 点击同级节点后，面包屑和详情同步刷新。
- [ ] 切换到 `分栏级联视图` 时：
  - [ ] 页面展示三列，列标题分别为 `一级`、`二级`、`三级`。
  - [ ] 点击一级节点后，二级列刷新为其子节点。
  - [ ] 点击二级节点后，三级列刷新为其子节点。
  - [ ] 点击任意列节点后，右侧详情区展示该节点。
- [ ] 既有模式行为不退化：
  - [ ] `树+详情` 仍可正常浏览与编辑节点。
  - [ ] `树形表格` 仍可展开并执行上下移动/启停操作。
- [ ] 前端测试通过：`npm run test:run`。
- [ ] 前端构建通过：`npm run build`。
