# 移动端聊天列表详情分离验收标准

## 范围
- 用户端聊天页：`/chat`、`/chat/[conversationId]`
- 企业端聊天页：`/enterprise/chat`、`/enterprise/chat/[conversationId]`

## 功能验收
1. 移动端访问列表页时，仅显示会话列表，不显示聊天详情区域。
2. 移动端访问详情页时，仅显示聊天详情，不显示会话列表。
3. 详情页左上角存在“返回会话列表”入口，且跳转到各自列表路由。
4. 在移动端列表页点击会话后，可进入对应会话详情路由。
5. 图片消息缩略图在移动端不会超出消息容器边界。
6. 聊天页移动端内边距较原实现更紧凑，视觉留白明显减少。

## 回归验收
1. 桌面端聊天双栏结构仍可用。
2. 用户端与企业端发送/加载消息基础流程未被本次改动破坏。
