# 登录过期强制退出验收标准

## 功能验收
- [ ] 当接口返回 HTTP 401 时，前端清理登录态并跳转登录页。
- [ ] 当接口返回 HTTP 200 + `{code:401}` 时，前端同样清理登录态并跳转登录页。
- [ ] 用户端/企业端跳转到 `/login`。
- [ ] 管理端跳转到 `/admin/login`。

## 质量验收
- [ ] 新增单元测试覆盖包装响应 `code=401` 的强制登出行为。
- [ ] 前端构建通过：`npm run build`。
- [ ] 前端测试通过：`npm run test:run`。
- [ ] 后端编译通过：`mvn compile`。
- [ ] 后端测试通过：`mvn test`。
- [ ] 使用 CDP 浏览器验证过期场景下会跳转登录页。
