# 管理端标签管理真实字段对接验收标准

## AC-1 字段对齐
- 标签表格不再展示“分类”列。
- 标签表格不再展示“引用次数”列。
- 列表数据来源仅依赖真实技能表可用字段（名称、同义词）。

## AC-2 筛选区精简
- 顶部仅保留一个搜索输入框。
- 不再出现“分类”筛选、“状态”筛选和“筛选”按钮。

## AC-3 同义词展示
- 当后端返回同义词数组时，页面展示每个同义词。
- 当后端同义词为空或 null 时，页面显示“无”。

## AC-4 操作按钮
- 每行右侧必须存在“编辑”按钮。
- 每行右侧必须存在“删除”按钮。
- 删除点击后需有确认流程并调用删除接口。

## AC-5 自动化测试
- 至少一条页面测试覆盖 AC-1~AC-4 的核心行为。
- 现有 null-safe 测试保持通过。
