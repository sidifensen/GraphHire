# 企业端手机版原版对齐验收标准

## 功能验收
- [ ] `mobile-enterprise` 首页、职位、职位详情、发布、编辑、推荐、团队、消息页面视觉结构与原版一致（含 375 宽容器布局）。
- [ ] 推荐页每个候选人卡片提供“详情”入口，并可进入候选人详情页面。
- [ ] 新增候选人详情页后，页面具备返回、匹配分析、技能与经历展示区域。

## 代码验收
- [ ] 移动端企业壳组件与导航不再使用“全宽版”断言对应实现。
- [ ] 相关测试已更新为原版样式预期并通过。

## 验证验收
- [ ] 前端构建通过：`npm run build`
- [ ] 前端测试通过：`npm run test:run`
- [ ] 后端编译通过：`mvn compile`
- [ ] 后端测试通过：`mvn test`
- [ ] 使用 CDP 浏览器验证 `mobile-enterprise` 关键页面可访问且样式正常。
