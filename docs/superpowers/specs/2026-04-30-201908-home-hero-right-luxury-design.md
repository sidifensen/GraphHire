# 首页 Hero 右侧轻奢重设计规格

- 日期：2026-04-30
- 需求来源：用户反馈“右侧设计太丑”，指定方向为“轻奢品牌感”。

## 目标

在不改动左侧主文案与 CTA 的前提下，重做首页 Hero 右侧视觉模块，提升品牌高级感与信息层次。

## 设计方向

采用“轻奢品牌感”：
- 主视觉大图作为情绪底图
- 半透明玻璃态浮层承载核心指标
- 底部数据条承载业务指标，形成“可读+可看”组合

## 结构与交互

1. 桌面右侧容器高度提升为 520px，增强视觉存在感。
2. 增加外层柔光渐变晕染，营造高端品牌氛围。
3. 主图上叠加深色渐变和径向高光，保证文字可读性。
4. 左上新增玻璃态卡片：`AI匹配指数`、指数值、说明文案。
5. 底部新增三列玻璃态指标条：`企业活跃席位`、`候选人响应率`、`7日成功匹配`。

## 约束

- 保持现有技术栈（React + Tailwind + framer-motion）。
- 不新增依赖。
- 同步改动 `src/app/page.tsx` 与 `src/features/mock-user/pages/Home.tsx`，避免双实现视觉漂移。

## 测试策略

- 页面单测新增对右侧关键文案的断言：
  - `AI匹配指数`
  - `企业活跃席位`
  - `候选人响应率`
- 浏览器实机检查首页 Hero 首屏表现与可读性。
