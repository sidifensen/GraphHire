# GraphHire 移动端项目生成提示词（用于 AI 线上构建应用）

你是一个资深全栈工程师。请根据我提供的移动端原型图，直接生成一个可运行的前后端项目（不是静态原型），并完成基础业务流程。

## 一、项目目标
构建“GraphHire 图谱智聘”移动端 Web 应用（PWA 风格），优先还原我提供的原型图视觉与交互。

## 二、技术栈要求
1. 前端：Next.js（App Router）+ TypeScript + Tailwind CSS
2. 状态管理：Zustand
3. 请求层：Axios
4. 表单：React Hook Form + Zod
5. 图表/图谱：react-force-graph-2d（用于我的图谱页）
6. 后端：Node.js + NestJS（或 Next.js Route Handlers 也可，但需分层清晰）
7. 数据库：PostgreSQL（Prisma ORM）
8. 缓存：Redis（可先接入基础封装）
9. 鉴权：JWT（用户端/企业端角色区分）
10. 全部页面文案必须中文

## 三、必须实现的端与页面（按现有原型）

### A. 用户端（移动端）
底部 Tab：首页 / 职位 / 公司 / 我的

页面：
1. 首页

3. 职位详情（智能匹配 + 投递）

5. 公司详情
6. 我的（上方个人信息卡 + 下方菜单列表）
7. 个人资料
8. 简历管理
9. 简历上传
10. 投递记录
11. 我的图谱
12. 通知中心

### B. 企业端（移动端）
底部 Tab：工作台 / 职位 / 推荐 / 团队

页面：
1. 工作台

5. 通知支持：列表、标记已读、全部已读
