# AI应用构建提示词

请根据我提供的手机端原型图，直接生成一个可运行的 GraphHire 移动端网站。

要求：
1. 严格按原型图还原页面结构与视觉风格，所有文案使用中文。
2. 处理好所有页面跳转关系，保证从列表到详情、从菜单到子页面的路径完整可用。
3. 页面切换要自然流畅，包含基础动效（如淡入、滑动、按钮反馈），提升交互体验。
4. 页面内容需要有真实动态交互（如筛选、搜索、状态切换、通知已读等），不是纯静态展示。
5. 页面滚动体验要顺滑，列表和卡片在触摸/点击时要有明显反馈动效。
6. 所有按钮和可点击区域都要有交互动效（按下、悬停/触摸反馈、禁用态）。
7. 加载过程要有清晰的加载动效（如骨架屏或 loading 状态），避免页面空白。
