# 企业端手机端 UI 原型提示词（重点前置版，可直接投喂 AI）

请为“GraphHire 图谱智聘-企业端”生成一套高保真手机端 UI 原型（iPhone 14 尺寸，375x812，浅色主题，专业管理后台风格，蓝色主色，卡片化布局）。

## 重点要求（必须优先满足）
1. 所有页面必须使用中文（页面标题、菜单文案、按钮文案、表单提示、状态提示、错误提示全部为中文）
2. 底部 Tab 栏固定为：`工作台 / 职位 / 推荐 / 团队`
3. 页面需可点击跳转，输出完整页面流和跳转关系
4. 移动端单手操作优先：核心按钮采用底部吸附操作区

## 页面与跳转关系
1）工作台（Tab：工作台）
- 核心指标卡片：待处理投递、新匹配候选人、在招职位
- 顶部右上角通知图标（进入通知中心）
- 快捷入口：
  - 发布职位 -> 新增职位页
  - 查看智能推荐 -> 候选人推荐页
  - 员工管理 -> 员工管理页
- 近期职位列表，点击职位 -> 职位详情

2）职位管理（Tab：职位）
- 状态筛选：全部/已发布/草稿/已关闭
- 搜索栏：职位关键词
- 职位卡片列表（职位名、部门、城市、薪资、曝光量、投递数、高匹配数）
- 卡片操作：
  - 详情 -> 职位详情页
  - 匹配候选人 -> 推荐页（带 jobId）
  - 发布/下架（页内操作）
- 页面主按钮：发布新职位 -> 新增职位页

3）新增职位
- 表单：职位名称、城市、最低薪资、最高薪资、职位描述
- 主按钮：创建并发布
- 提交成功 -> 回到职位管理（带成功提示）

4）职位详情
- 展示职位完整信息、技能要求、状态
- 按钮：
  - 修改职位 -> 编辑职位页
  - 查看匹配候选人 -> 推荐页（带 jobId）
  - 一键匹配全部候选人（页内执行）

5）编辑职位
- 表单回填
- 保存修改 -> 返回职位详情

6）候选人推荐（Tab：推荐）
- 顶部职位选择器（切换职位刷新推荐）
- 搜索候选人
- 推荐列表卡片（候选人、匹配分、匹配理由、技能/经验/学历分）
- 操作：
  - 详情 -> 候选人详情页
  - 邀请 -> 面试安排页
  - 一键匹配候选人（页内执行）
  - 刷新（页内）

7）候选人详情（新增）
  - 完整简历、匹配解释、技能匹配、操作记录
  - 按钮：发送面试邀请、返回推荐列表

8）员工管理（Tab：团队）
- 统计卡：企业总人数、企业主、管理员
- 待加入审批列表（通过/拒绝）
- 员工列表（账号、角色、状态、最近登录）
- 操作：添加员工、重置密码、禁用/启用

9）通知中心（由顶部右上角通知图标进入）
- 分类筛选：全部/新推荐/简历投递/认证反馈/系统公告
- 列表卡片（标题、时间、内容、未读标识）
- 操作：标记已读、全部已读、清空已读
- 通知关联跳转：
  - 推荐类 -> 候选人推荐
  - 投递类 -> 职位详情或候选人详情

## 输出格式要求
- 输出：页面清单 + 页面跳转说明 + 关键交互标注
- 输出一套可点击原型（Prototype）
